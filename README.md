目前打算开发的Demo版本应当包括以下功能：

    1.界面：状态栏(指示设备连接状况)、动作显示窗口、按钮（动作选择、动作录制）
    
    2.动作录制：提取示例动作特征并保存（之后应添加处理功能，切除录制首尾的无效动作）
    
    3.动作对比：使用DTW算法，输出bool值。
